A COMPLETE PACKAGE

This is a complete package that can help automation of record keeping in small libraries. It keeps track of books and members of library. Allows issuing of a book to certain user, returning book, checking availability of the book ,managing book and member information. It also allows you to save the records to a file and load them back into memory of the library manager. Allows user to view edit and delete records of books and members.
